# Collazos-Macroboard-MK1
This is the first Macroboard that i have made, and this made me interested on the all the hack club stuff, hope this project is just the beggining of a path with Blueprint
## Bom

- 9x Cherry MX Switches
- 3x SK6812 MINI Leds
- 1x XIAO RP2040
- 9x Blank DSA Keycaps

- 9x M3 Heatset
